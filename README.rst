=======
PyGlacier
=======

Es un proyecto Python, que sirve de Wrapper al API de Amazon Glacier: `Amazon Glacier <http://http://aws.amazon.com/es/glacier/>`_.



Requiere:

- Solo requiere la instalacion de la libreria boto (pip install boto)


Instalacion:
------------
